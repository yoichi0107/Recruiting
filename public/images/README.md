# 画像の配置場所

このディレクトリに、以下のファイル名で画像を配置してください。
コードはこのファイル名を参照しています。画像が無い場合は、各 `<Image>` の
プレースホルダ背景（`#E5E3DC`）が表示され、レイアウトは崩れません。

| ファイル名 | 用途 | サイズ目安 |
|---|---|---|
| `107design-logo.png` | ヘッダー・フッターのロゴ | 透過 PNG / 横 240px 程度 |
| `hero-main.jpg` | Section 1 ファーストビュー背景 | 横 2400px / 抽象・知的・モノトーン寄り |
| `culture-01.jpg` | Section 2 チーム・歴史を感じる写真 | 横 1600px / モノクロまたは彩度低め |
| `company-overview.jpg` | Section 3 会社紹介の雰囲気写真 | 横 1600px |
| `work-scope.jpg` | Section 4 仕事の射程（任意・未使用でも可） | 横 1600px |
| `nishikawa-portrait.jpg` | Section 8 代表ポートレート | 縦長 800×1000px 推奨 |
| `og-image.jpg` | OGP用 | 1200×630px |

> 写真はモノクロ／彩度低めで統一感を出す方針です（コード側で `grayscale` を適用済み）。
